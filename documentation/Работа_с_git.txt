1) git clone ccылка
2) Создаем ветку: git checkout -b TASK-1

4) Коммитем: git commit -m "сделал 1 задачу"
5) Загружаем в ветку изменение, если ветка создана 1 раз: 
	git push --set-upstream origin TASK-1
