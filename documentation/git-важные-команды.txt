Список полезных команд:
								(символы <> - не пишут при написании в терминале или cmd команды, например, git add test.txt)
git status						зелёным показывает файлы в commint, а красным не добавленные
git add <file_name>				Добавляет 1 файл (можно в "" указывать через пробел несколько)
git add .						Добавляет все изменённые файлы
git restore --staged <file>		удаление файла из буфера, если добавили не тот
git commit -m "your comment"	Создаём коммит для подготовке к отправке. В "" пишут комментарий к добавленным файлам
git push						Отправляем созданный коммит на репозиторий
git log							Отображет информацию о коммитах, автор, дата загрузки, комментарий.
git pull						Скачиваем из репозитория себе изменённые файлы
git branch 						Отображет все ветки, перед текущей веткой находится символ *
git branch <name>				Создание новой ветки с именем name
git checkout <name>				Переключиться на ветку name
git chechout -b <name>			Создание новой ветки в репозитории и мгновенное переключение на неё
